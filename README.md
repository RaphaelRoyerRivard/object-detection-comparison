# faster-rcnn-test

For `faster_rcnn_test.py` and `dense_optical_flow.py` to work, your directory tree should look like that :

	- Some folder
		- `object-detection-comparison`
		- `images`
			- dataset1
			- dataset2
			- etc.

Result files in `Analyse` have the following information :

	- Precision
	- Recall
